Users Route Description

Method Path Description
FetchUsers /users get: Gets All Users
FetchUser /users/:id get: Gets a Single User
CreateUser /register post: A User
LoginUser /login post: A User
UpdateProduct /users/:id put: Update a User
DeleteUser /users/:id delete: Delete a User

Product Route Description

Method Path Description
FetchProducts /products get: Gets All Products
FetchProduct /products/:id get: Gets a Single Product
AddProduct /products post: Adds a Product
UpdateProduct /products/:id put: Update a Product
DeleteProduct /products/:id delete: Delete a Product
